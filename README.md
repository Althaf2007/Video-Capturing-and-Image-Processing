# Video-Capturing-and-Image-Processing
# Aim
To capture images and process live video frames using the OpenCV library in Python. The experiment demonstrates image capturing, video streaming, resizing, and rotating video frames.
Requirement
Hardware Requirements
Laptop/Desktop with Webcam
Minimum 4 GB RAM
Software Requirements
Python 3.x
OpenCV Library
Matplotlib
Jupyter Notebook / Google Colab
Concept
OpenCV (Open Source Computer Vision Library) is widely used for image processing and computer vision applications.
In this experiment:
The webcam is accessed using OpenCV.
A single image frame is captured and stored.
Live video frames are displayed continuously.
Frames are resized to different dimensions.
Video frames are rotated clockwise.
Matplotlib is used for displaying images.
This experiment helps in understanding real-time image acquisition and basic video processing techniques.
Procedure
Import required libraries such as OpenCV, Matplotlib, and Time.
Access the webcam using VideoCapture().
Capture a single image frame and save it.
Display the captured image.
Read live video frames continuously.
Convert frames from BGR to RGB format.
Resize the video frames.
Rotate the frames clockwise.
Display processed frames continuously.
Release the webcam after execution.
Output
1. Captured Image Output
The webcam captures a single frame.
The captured image is displayed successfully.
2. Live Video Streaming Output
Real-time webcam video is displayed continuously for 50 frames.
3. Resized Frame Output
The original frame is resized to smaller dimensions.
The resized video stream is displayed successfully.
4. Rotated Frame Output
The live webcam frames are rotated 90° clockwise.
The rotated video stream is displayed continuously.
Result
Thus, the experiment on Image Capture and Video Processing Using OpenCV was successfully implemented using Python and OpenCV. Image capturing, live video streaming, frame resizing, and frame rotation operations were performed successfully.
